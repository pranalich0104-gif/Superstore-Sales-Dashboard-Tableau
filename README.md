# 📊 Superstore Sales Dashboard - Tableau

## 📌 Project Overview
This project is an interactive Tableau dashboard built using the Sample Superstore dataset. It provides insights into sales, profit, customer segments, shipping modes, regional performance, product categories, and discount impact on profitability.

---

## 📷 Dashboard Preview

![Dashboard](Dashboard%20Image.png)

---

## ✨ Features

- KPI Cards (Sales, Profit, Orders, Customers)
- Interactive Filters
- Sales by Category
- Sales by Sub-Category
- Profit by Region
- Monthly Sales Trend
- Customer Segment Distribution
- Ship Mode Distribution
- Top 5 Products by Profit
- Discount vs Profit Analysis

---

## 🛠 Tools Used

- Tableau Desktop
- Sample Superstore Dataset

---

## 📂 Dataset

Sample Superstore Dataset

---

## 📈 Key Business Insights

- West Region generates the highest profit.
- Office Supplies has the highest sales.
- Consumer Segment contributes more than 50% of customers.
- Standard Class is the most preferred shipping mode.
- Higher discounts generally reduce profitability.

## 🚀 How to Open

1. Download the repository.
2. Open **SuperStore Dashboard.twbx** using Tableau Desktop.
3. Explore the dashboard using interactive filters.
